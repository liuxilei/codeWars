### 题目
> Multiply

### 描述
> The code does not execute properly. Try to figure out why.

### Example
```javascript
function multiply(a, b){
    a * b
}
```

### My Solution
```javascript
function multiply(a, b){
    return a * b
}
```

### Best Practices
```javascript
multiply = function (a, b) {
    return a * b;
}
```
